# ISMISM

基于rust的也许跨平台的快速打开主义主义视频的命令行程序

## 兼容性

|平台|是否兼容|
|:-:|:-:|
|Windows 10/11|是|
|Linux(Ubuntu 22.04)|未测试|
|MacOS|未测试|

## 使用

**请先在deserialize()与serialize()函数中修改文件路径！！！**

ismism.exe [args]

Args:

- -a: 添加条目
- -m [ismism]: 修改条目
- -o [ismism]: 打开条目
- -f [regex]: 搜索条目,支持正则表达式
- -l: 列举条目
- -h: 帮助
